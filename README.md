# 💰 MoneyMate

MoneyMate is a mobile-first **personal finance app** designed to help users build **consistent financial habits**, not just track expenses.

Unlike traditional expense trackers, MoneyMate focuses on **behavior, awareness, and proactive budgeting** to solve the real problem:  
👉 People don’t track expenses consistently.

> 🚧 This project is currently under active development.

---

## 🚧 Current Features (MVP - In Progress)

- 🔐 **Authentication**
  - User login/signup using Supabase Auth

- 💸 **Income Setup**
  - Users can set and store their monthly income

- 📊 **Basic Data Flow**
  - Core structure for storing financial data in Supabase

- 🧪 **Initial Screens**
  - Auth flow
  - Income entry flow
  - Debug/home screen for data testing

---

## 🛠️ Tech Stack

- **Frontend:** React Native (Expo), JavaScript  
- **Backend:** Supabase (PostgreSQL + Auth)  
- **Notifications:** Firebase Cloud Messaging (planned)  
- **Native Integration:** Android SMS parsing module (planned)  

---

## 🎯 Problem Statement

Most users:
- Don’t track expenses consistently  
- Forget where money is spent  
- Lack awareness of budget deviations  

👉 Existing apps focus on logging, not behavior.

---

## 💡 Solution

MoneyMate aims to:
- Make expense tracking **effortless**
- Encourage **habit formation**
- Provide **insight-driven feedback**
